# Quant Lab 06 — 机器学习交易策略

一份可交互的量化研究作品：从财务因子衍生、滚动样本外分类，到季度选股组合与成本后回测。

## 研究摘要

- 数据：20,733 条股票—季度样本，4,280 只股票
- 特征：17 项原始财务指标，衍生为 21 项模型因子
- 模型：逻辑回归、决策树、随机森林
- 目标：下一季度收益是否高于当季横截面中位数
- 策略：预测概率前 20% 等权持有，季度调仓，单边成本 0.10%
- 样本外：3 个季度，逻辑回归 AUC 0.632
- 结果：逻辑回归累计 -3.0%，同期全市场等权 -19.8%

测试期市场整体下跌，因此本实验更接近一次“减亏能力”检验，而不是盈利证明。项目主动披露短样本、简单价格收益、未计股息等边界。
